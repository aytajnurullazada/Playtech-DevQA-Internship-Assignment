# Playtech DevQA Internship Assignment

Selenium WebDriver tests for playtechpeople.com

## What it does
1. Opens the website
2. Finds and lists all teams in the company
3. Lists research areas for reducing gambling-related harm
4. Prints job links for positions in Estonia (Tartu and Tallinn)
5. Closes the browser

## Bonus
- Results exported to test_results.txt
- Tests run via JUnit 5

## How to run
Requires Java 17+, Maven, and Chrome. Then: mvn test

## Built with
- Java
- Selenium WebDriver
- JUnit 5
- WebDriverManager
